# Agartha Ecosystem

### Where Digital Reality Meets Human Consciousness

Agartha is a revolutionary digital ecosystem unifying augmented reality (AR), cannabis culture, music creation, and AI companionship. This ecosystem brings together Weedland, Music Machine, and MyAI, using blockchain, AR/VR, and AI technologies to enhance creative expression and community building. Powered by **NEAR Protocol** and **The Matrix AI**, Agartha offers a seamless experience that transcends traditional digital platforms.

## üöÄ Getting Started

### Prerequisites
- **Node.js** and **npm** (for backend and frontend dependencies)
- **MetaMask** or any other Ethereum-compatible wallet (to interact with blockchain features)
- **NEAR Wallet** (for blockchain transactions via NEAR Protocol)
- **Meta Quest Glasses** (optional for AR features)

### Installation

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/your_username/Agartha-Ecosystem.git
    cd Agartha-Ecosystem
    ```

2. **Install Dependencies**:
    - For the backend:
      ```bash
      cd backend
      npm install
      ```
    - For the frontend:
      ```bash
      cd ../frontend
      npm install
      ```

3. **Configure Environment Variables**:
    - Copy `.env.example` to `.env` and add the relevant API keys and configurations:
    ```bash
    cp .env.example .env
    ```

4. **Run the Development Servers**:
    - Backend:
      ```bash
      cd backend
      npm start
      ```
    - Frontend:
      ```bash
      cd ../frontend
      npm start
      ```
